Why?

Ditch the pain and agony

Adding file uploads to your application is a pain. To do it yourself at a minimum you will need to:

  • Securely configure a cloud storage service
  • Set up IAM credentials
  • Configure your client to send direct uploads
  • Create presigned URLs on your server
  • Configure your bucket CORS
  • Implement client and server-side validations

Oh and don’t forget that you’ll need to make it look good and match the rest of your application. You need uploading, but you don’t have time for that headache or want to manage that much infrastructure.

A Better Way

What if you didn’t have to do these things? What if you could just drop in a line of code, create a few file inputs, and have the files magically uploaded to the cloud for you? Simple File Upload can help you get there. With our product you will save time and headache.

Enter Simple File Upload

Simple File Upload takes care of all the heavy lifting surrounding file uploads. The widget hooks into an input with class of simple-file-upload and replaces it with a dropzone. The user drops a file and the file is automatically uploaded to cloud storage via a direct upload. Files are served via a CDN which is imperitive in today's modern

Will it work for me?

Yes! Simple File Upload can be installed via a javascript snippet or React component. It works will all backend languages and databases.
